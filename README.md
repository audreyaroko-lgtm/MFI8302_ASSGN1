
# Monte Carlo Simulation for European Options

This repository contains an implementation of Monte Carlo Simulation in R for pricing European Call and Put Options.

## Course

MFI 8302 Computational Methods in Finance

## Contents

- Monte_Carlo_European_Options.R

## Model

The simulation assumes that stock prices follow Geometric Brownian Motion under the risk-neutral measure.

The Monte Carlo estimates are compared with the Black-Scholes analytical solution.
